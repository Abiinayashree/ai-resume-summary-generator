# AI Resume Summary Generator

## Project Overview
Built an AI-powered resume summary generator using Python and Groq API. The application generates professional resume summaries dynamically based on user-provided skills and experience.

## Technologies Used
- Python
- Groq API
- Llama 3.1 Model

## Key Features
- AI-generated professional summaries
- Dynamic text generation
- Prompt-based content generation
- User skill analysis

## Concepts Learned
- Prompt engineering
- AI text generation
- API integration
- Large Language Model workflow
- Dynamic content generation

## Project Workflow
1. User enters skills and experience
2. Prompt is generated dynamically
3. Groq API processes the prompt
4. AI-generated professional summary is displayed
